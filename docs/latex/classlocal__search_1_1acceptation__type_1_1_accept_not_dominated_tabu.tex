Clase que mantiene un frente de Pareto con estrategia tabú 

Esta clase implementa una estrategia de aceptación para problemas multiobjetivo que mantiene y actualiza el frente de Pareto de soluciones no dominadas, incorporando el concepto de búsqueda tabú.
