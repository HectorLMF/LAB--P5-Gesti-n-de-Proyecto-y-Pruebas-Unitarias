\doxysubsection{Descripción detallada}
Clase que acepta candidatos basándose en temperatura (Simulated Annealing). 

Esta clase implementa el criterio de aceptación de Simulated Annealing, donde soluciones peores pueden ser aceptadas con una probabilidad que depende de la temperatura actual del sistema. Permite escapar de óptimos locales.
