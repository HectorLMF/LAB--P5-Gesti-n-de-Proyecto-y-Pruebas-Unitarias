\doxysubsection{Descripción detallada}
Clase que implementa el método de factores ponderados para evaluar estados en problemas multiobjetivo. 

Esta clase combina múltiples funciones objetivo mediante pesos asignados a cada una, produciendo una única evaluación agregada del estado.
