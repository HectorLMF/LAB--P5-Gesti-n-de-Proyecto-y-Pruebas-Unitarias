\doxysubsection{Descripción detallada}
Clase abstracta que define operadores para generar nuevos estados. 

Esta clase proporciona la interfaz para operadores que pueden generar nuevos estados a partir de estados existentes o de forma aleatoria. Es la base para operadores de búsqueda local y generación aleatoria.
