\doxysubsection{Descripción detallada}
Coordinador central para la ejecución de algoritmos metaheurísticos. 

Esta clase singleton gestiona la estrategia global de optimización, coordinando la ejecución de diferentes generadores metaheurísticos, manteniendo el seguimiento del mejor estado encontrado, gestionando las iteraciones y recopilando estadísticas de rendimiento. Soporta múltiples modos de operación incluyendo algoritmos de un solo generador y portafolios de múltiples generadores (\doxylink{classmetaheuristics_1_1generators_1_1_multi_generator}{Multi\+Generator}).

La clase también gestiona entornos dinámicos donde el problema puede cambiar durante la ejecución, permitiendo adaptación y reinicio de referencias.
