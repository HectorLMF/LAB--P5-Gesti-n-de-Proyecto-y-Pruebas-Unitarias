\doxysubsection{Descripción detallada}
Representa el estado de una solución en un problema de optimización. 

Esta clase encapsula la información de un estado o solución, incluyendo su codificación, evaluación, número identificador y tipo de generador. Proporciona métodos para comparar, copiar y calcular distancias entre estados.
