\doxysubsection{Descripción detallada}
Clase que gestiona la actualización de parámetros y cambio de generadores. 

Esta clase permite la actualización dinámica de parámetros durante la ejecución y el cambio automático entre diferentes generadores metaheurísticos según el número de iteraciones.
